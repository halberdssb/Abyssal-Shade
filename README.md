# Enemy Stalking

## Unity Version: 2022.3.62f3

## Files I was Primary Author For: Enemy Stalking.cs, Enemy Chasing.cs, and Angler Attract.cs

## C# File: Enemy Stalking (Assets -> Scripts -> Enemy Stalking.cs)

    This is an old file written in C# from an old game I made with a group of friends for class during my junior year, the game being Abyssal Shade. This was my first time being given the freedom to work on code to program the AI of enemy characters in the game. I chose the Enemy Stalking file because despite it's age, it is the one I am most proud of and believe it still holds up despite its age. It allowed our sea serpent enemy to switch between a stalking mode, where it would track the player's position and slowly circle around them, and lunging mode where it would lunge towards the player to attack them, before swiftly returning to its circling position to continue stalking the player. A more detailed explanation can be found in the .cs file.
